# DijkstraFlow

Interactive Dijkstra algorithm visualizer with custom graphs, live distance tables, and step-by-step playback.

## Features

- Interactive editor: click to add nodes, shift-drag to create weighted edges
- Playback controls: run, resume, step, reset
- Live distance table and execution log
- Random graph generation with configurable node count

## Quick Start

```powershell
npm install
npm run dev
```

## Build

```powershell
npm run build
```
